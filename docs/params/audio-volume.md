# Audio Volume

Changes the audio volume.

Expected value: a volume number in dB.

- Default: `original` - keep volume unchanged.
- Negative value decreases the volume. Minimal value is -50 (dB).
- Positive value increases the volume. Maximal value is 50 (dB).
- `0` resets the value to `original`.
