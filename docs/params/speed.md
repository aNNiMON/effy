# Speed

Changes the video speed and the audio tempo.

Expected value: a decimal speed/tempo number in range 0.5...100.

- Values lesser then 1 - slow down a video/audio.
- Values greater then 1 - speed up a video/audio.
- Default: `1` - don't change a speed/tempo.
